# PeekVision — Screening Demo

This is a demo project that showcases a web application for screening retina photos. The application allows users to upload a retina photo, and it performs a basic analysis of the image quality.

The project is divided into two main parts:

*   **Frontend:** A React application that provides the user interface for uploading and viewing the analysis report.
*   **Backend:** An Express.js server that stores the analysis reports.

This project is for demonstration purposes only and is not intended for clinical use.
